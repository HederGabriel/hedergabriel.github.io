# Portfolio - Heder Gabriel

Portfolio pessoal de desenvolvedor, hospedado no GitHub Pages.

## Sobre

Sou um desenvolvedor em formação, técnico em Informática e estudante de Tecnologia em Sistemas para Internet. Apaixonado por tecnologia e desenvolvimento de software.

## Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS, animações e efeitos)
- JavaScript (ES6+)
- GitHub Pages & GitHub Actions

## Estrutura do Projeto

```
portfolio/
├── index.html      # Página principal
├── styles.css      # Estilos e animações
├── main.js         # Interatividade e efeitos
├── .github/
│   └── workflows/
│       └── deploy.yml  # CI/CD para GitHub Pages
└── README.md
```

## Funcionalidades

- Design responsivo
- Partículas interativas com Canvas
- Cursor customizado
- Efeitos de tilt 3D nos cards
- Animações de scroll reveal
- Efeito de digitação (typing)
- Navbar com comportamento de hide/show
- Menu mobile
